## Ejemplo de código en Python

```python
import math

def area_circulo(radio):
  area = math.pi * radio ** 2
  return area

radio = 5
print(f'El área de un círculo con radio {radio} es: {area_circulo(radio)}')
// El resultado de ejecutar este código en la consola sería:
// El área de un círculo con radio 5 es: 78.53981633974483
```
